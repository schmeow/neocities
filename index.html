Projects

Glove Hand Controller:
I'm currently working on creating a glove which can track hand movements and manipulate a digital 3D model using arduinos and C++.

Automatic Plant Watering System: I created a arduino powered system that automatically waters plants by testing the water level of the soil, and sprinkles water when needed based on the plant type.

Analyzing Peg Solitare Game States using Wolfram Mathmatica and Multiway Graphs: During the Wolfram Summer Research Program in 2023, I did research on Peg Solitare, a single player game about pegs. I analyzed different board states and come up with algorithms to generate movesets.

Analyzing Walkability using Wolfram Mathmatica and OSM Data: During the Wolfram Emerging Leaders Program in 2023/2024, I worked with two other students to create a program that could analyze walkability in a given area. We created our own walkability model, and tested it on many different cities and towns.
