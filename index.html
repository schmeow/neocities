Glove Controller

Since the start of the school year, I've been working to create a fully internal glove controller. A glove controller is a type of controller often used for vr which measures hand and finger positions to control something digitally. Usually, they are made with external cameras and setup, but I wanted to create something fully internal using arduinos.

So far, I've been able to set up a gyroscope and arduino to read hand rotation and report it back to my computer. I'm currently working on creating a visualization of this input using OpenGL, and I plan to incorporate finger data in the future.

Drone

This year, our Aerospace Club has decided to build a drone! As one of the Presidents of the club, I've taken a role as the head of Electronics, working to plan out which motors, escs, arduinos, and other components we will need.
